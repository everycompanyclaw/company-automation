## What Zapier Does

Zapier is the market leader in no-code workflow automation, connecting 7,000+ web apps through a trigger → action model called "Zaps." Non-technical users build automations by selecting a trigger app and one or more action apps, with optional filters and formatters in between. It's the OG of consumer/small-business automation, widely known and trusted.

**Core Products:**
- Zapier Free (limited)
- Zapier Starter / Professional / Team / Company
- Zapier Tables (Airtable-like data store, launched 2024)
- Zapier Canvas (workflow visualization, launched 2024)
- Zapier AI (natural language Zap building, launched 2024)

## 🔴 Key Weaknesses & Complaints (2025–2026)

### 1. **Task-Based Pricing Gets Expensive Fast**
Multi-step Zaps multiply task consumption. A 4-step Zap running 100x/day = 300 tasks per run × 30 days = 9,000 tasks/month — already off Starter tier. Businesses at 50,000+ tasks/month pay $300–600/month.

### 2. **October 2025 Outage**
Major service disruption caused widespread workflow failures. No SLA credits for most plans. Eroded trust among power users.

### 3. **March 2025 Security Breach**
Unauthorized user accessed code repositories. Customer data was "inadvertently copied to repositories for debugging purposes." Public disclosure was delayed. Significant reputation damage.

### 4. **Trustpilot: 1.7/5 from 190+ reviews**
Polarized reviews — users love integrations but consistently complain about:
- Expensive pricing at scale
- Strict no-refund policy
- Task overages billed unexpectedly
- Customer support response times

### 5. **Slow Feature Development**
Power users criticize Zapier for lagging behind Make.com on:
- Visual workflow canvas (Make has had this for years)
- Looping capabilities (Zapier only added loops in 2024)
- Conditional branching complexity
- Error handling granularity

### 6. **Limited Code/Custom Logic**
Unlike n8n or Pipedream, Zapier doesn't let users run custom code easily (requires Professional+ and limited to Code steps). No native JavaScript/Python execution.

### 7. **Premium App Gating**
Major connectors (Salesforce, HubSpot, QuickBooks) require paid plans even for basic triggers. Anti-competitive bundling.

## 🎯 Competitive Positioning for EveryCompanyClaw

**Zapier's pain points we exploit:**
1. "Zapier bills you per task — we bill you per month, unlimited"
2. "Zapier got hacked in 2025 — we run on your infrastructure"
3. "Zapier locks custom code behind Professional — it's free here"
4. "Zapier can't loop or branch well — we were built for complex workflows"

**Go after:** Zapier users burning $200+/month, technical teams who feel limited by Zapier's no-code constraints, indie hackers and bootstrapped startups.

**Avoid:** Non-technical users who only need simple email-to-slack Zaps — Zapier wins on pure simplicity for ultra-basic use cases.
