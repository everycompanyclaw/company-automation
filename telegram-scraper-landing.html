Questions

Do I need a Telegram account? +

Yes — you need a Telegram app via my.telegram.org to get your API credentials (hash + ID). Takes 2 minutes and your phone number stays private. The script uses the official MTProto API, not a bot token.

Can I scrape private channels? +

No — the tool only works with public channels, as per Telegram's Terms of Service. This is actually a feature: it means you're always working with publicly available data, keeping your usage well within Telegram's policies.

Is this a subscription? +

No. Pay once, own it forever. Pro at $49 includes all current features and every future update. Enterprise at $199 adds white-label rights and custom integrations. No caught trials, no recurring billing.

How does delivery work? +

After purchase, you'll receive download links + setup instructions via email within minutes. Check your inbox (and spam folder). Questions? Email everycompanyclaw@gmail.com.

What about Telegram's rate limits? +

Built-in flood-wait handling automatically backs off when Telegram rate-limits you, then resumes. No manual intervention needed. Larger scrapes are chunked to stay well within safe limits.

Can I use this for client work? +

Yes — the Enterprise license covers white-label use for client projects. Pro license covers your own internal use and products. Contact us for custom agency pricing if you need multi-seat deployment.

What Python version do I need? +

Python 3.8 or higher. Dependencies (Telethon and a few standard library modules) are listed in requirements.txt — pip install -r requirements.txt and you're ready.

What's the difference between this and a Telegram bot? +

Bots can only see messages sent to them. This script uses your personal Telegram account session (MTProto) to read any public channel you can access — giving you full historical data and real-time monitoring that bots simply can't do.
