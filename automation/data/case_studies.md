# Case Study: Law Firm Document Automation

## Client
Small law firm (5 attorneys), Hong Kong

## Problem
- 15+ hours/week manually processing client documents
- Repetitive intake forms
- Manual deadline tracking in spreadsheets

## Solution
- Python automation scripts for document parsing
- Zapier workflows for intake forms → CRM
- Automated deadline reminders via email

## Results
- **15 hours/week saved**
- **0 missed deadlines** since implementation
- **Estimated $3,000/month** value

## Tech Stack
- Python, Zapier, Google Sheets, Gmail

## Quote
> "Transformed our operations. Wish we'd done this years ago."
> — Managing Partner

# Case Study: E-commerce Inventory Alert System

## Client
Online retail store, 500+ SKUs

## Problem
- Manual inventory checks multiple times daily
- Stockouts causing lost sales
- No real-time visibility

## Solution
- Python script monitoring inventory API
- Slack alerts when stock < threshold
- Auto-reorder workflow

## Results
- **90% fewer stockouts**
- **2 hours/day saved** on manual checks
- **Estimated $5,000/month** in saved sales

## Tech Stack
- Python, Shopify API, Slack

# Case Study: Marketing Agency Reporting

## Client
Digital marketing agency, 20+ clients

## Problem
- 10+ hours/week building client reports
- Data scattered across platforms
- Inconsistent formatting

## Solution
- Automated data collection from Google Ads, FB, Analytics
- Python generating PDF reports
- Scheduled delivery to client portals

## Results
- **10 hours/week saved**
- **Clients upgraded** due to professional reports
- **Scalable to 3x clients** without extra work

## Tech Stack
- Python, Looker Studio, Google Ads API, SMTP
