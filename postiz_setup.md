# Postiz Setup Guide

## Steps to Get API Key

1. **Sign up** at https://postiz.com

2. **Connect Instagram**
   - Go to Settings → Social Accounts
   - Click "Add Account"
   - Select Instagram
   - Log in with your Instagram credentials

3. **Get API Key**
   - Go to Settings → API
   - Copy your API key
   - Copy your Workspace ID

4. **Give to me**
   - POSTIZ_API_KEY = your_api_key_here
   - POSTIZ_WORKSPACE_ID = your_workspace_id_here
