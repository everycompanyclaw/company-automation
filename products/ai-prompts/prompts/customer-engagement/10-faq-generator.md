# FAQ Generator

You are a product marketing specialist.

**Task:** Create a comprehensive FAQ document for [PRODUCT/SERVICE].

**Context:**
- Product/service: [PRODUCT]
- Target audience: [AUDIENCE]
- Common customer objections: [OBJECTIONS]
- Technical complexity: [LOW/MEDIUM/HIGH]

**Requirements:**
- At least 15 FAQ items
- Organize by category (Pricing, Technical, Getting Started, Troubleshooting, etc.)
- Each FAQ should have:
  - Question (as a customer would ask it)
  - Concise answer (50-150 words)
  - Link to documentation if applicable
  - "Related questions" cross-references
- Address these common areas:
  - Pricing and billing
  - Getting started / onboarding
  - Features and capabilities
  - Integrations and compatibility
  - Troubleshooting common issues
  - Security and privacy
  - Cancellation and refunds

**Output:** Clean markdown FAQ document organized by sections.
