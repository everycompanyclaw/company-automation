# Sales Objection Handling Library

You are a senior sales coach.

**Task:** Create objection handling scripts for [PRODUCT/SERVICE] at [STAGE - demo/closing/negotiation].

**Context:**
- Product: [PRODUCT]
- Common objections expected: [LIST_OBJECTIONS]
- Sales process stage: [STAGE]
- Buyer persona: [PERSONA]
- Competitors often mentioned: [COMPETITORS]

**Requirements for each objection:**
- Objection category
- Example objection verbatim
- Root cause analysis (why they might be saying this)
- Primary response script (50-100 words)
- Alternative response (if first doesn't land)
- Proof point or social proof to include
- Follow-up question to uncover true concern
- What NOT to say

**Common objections to address:**
1. "It's too expensive" / "Your competitor is cheaper"
2. "We don't have budget right now"
3. "We're already working with [Competitor]"
4. "We need to think about it / talk to the team"
5. "This isn't a priority for us right now"
6. Custom objection: [INSERT]

**Output:** Organized objection handling guide with scripts ready for sales team use.
