Command-Line Interface

# Scrape 10,000 messages from a channel
$ python telegram_scraper.py scrape --channel techcrunch --limit 10000 --format json
✓ Fetched 10,000 messages (2024-01-01 → 2026-04-11)
✓ Saved to techcrunch_messages.json

# Monitor a keyword in real-time
$ python telegram_scraper.py monitor --channel ai_startups --keyword "funding" --interval 300
🔔 [14:32] "AI startup raised $50M funding round" — @techcrunch

# Find trending topics
$ python telegram_scraper.py trends --channel ai_startups --hours 24
Top keywords: funding (+312%), launch (+189%), partnership (+156%)
